## ✍🏻 Conventions

### 1. Git & Github

| Type     | Description                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------- |
| chore    | 프로젝트 초기 설정, 폴더 및 파일 구조 설정, 추가, 수정, 삭제, 패키지 매니저 환경설정, 의존성 업데이트, ... |
| docs     | 문서 수정                                                                                                  |
| feat     | \[HTML\] 마크업, \[CSS\] 스타일링, \[JS\] 기능 추가                                                        |
| fix      | 오류를 발생시키는 코드, 버그, 오타 등 수정                                                                 |
| refactor | 코드 개선                                                                                                  |

<br />

  - max-width: 767px

  - max-width: 1279px
